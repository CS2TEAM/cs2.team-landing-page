---

Esports team finding platform and social network for Counter-Strike 2, enabling aspiring players to find teams or create their own.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v22.13.1
```

```bash
npm -v
# 11.1.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
